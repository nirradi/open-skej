# Quick Start for LRPP projects

Based on Linux, React, Python, Postgres

## Requirements

Docker.

## Running it

```
cd deploy
docker compose up
```

## Locations

- Client is at 127.0.0.1:3000
- Server is at 127.0.0.1:8000

## What now?

Develop something. Server files go to ./server, Client files go to ./client
